Errorc back to ToC or Class ToC

IRI: https://w3id.org/pko#Error

This class represents an Error that can be encountered while executing an activity.
scope note
In the context of the Procedural Knowledge Ontology (PKO), this class represents an error that an Agent can encounter during a Procedure or Step Execution. An Error can be associated with a description, an error code, and an error cause. The Procedure may contain a (fallback) step to address such error.
is in domain of
error cause dp, error code dp
is in range of
addresses error op, has encountered error op
